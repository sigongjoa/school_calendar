# 01. 프로젝트 개요

## 프로젝트명

**node15_school_calendar** - 학사일정 통합 캘린더

## 배경 및 목적

학원을 운영할 때, 학생들이 여러 학교에서 오기 때문에 각 학교의 학사일정(시험기간, 공휴일, 개교기념일, 방학 등)을 매번 개별적으로 찾아봐야 하는 번거로움이 있다.

이 모듈은:
- 나이스(NEIS) 공공 API를 통해 학교별 학사일정을 **자동 수집**
- 여러 학교의 일정을 하나의 캘린더에 **통합 표시**
- 학교별 토글로 **켜고/끄기**하여 필요한 학교만 필터링
- 학교별 학사일정을 테이블 형태로 **상세 조회**

## 핵심 기능 (MVP)

### 기능 1: 통합 캘린더 뷰 (CalendarPage)
- 월간 캘린더에 선택된 학교들의 학사일정을 색상별로 표시
- 사이드바에서 학교별 토글 (체크박스) on/off
- 시험 기간은 상단 바(bar)로 연속 표시
- 공휴일, 학교행사, 방학 등 카테고리별 뱃지
- 일간/주간/월간 뷰 전환
- 이전달/다음달/오늘 네비게이션

### 기능 2: 학교별 일정 관리 (SchoolSchedulePage)
- 좌측 사이드바: 등록된 학교 목록
- 학교 선택 시 해당 학교의 학사일정 테이블 표시
- 날짜, 구분(시험/공휴일/학교행사/기타), 이벤트명, 상태
- 나이스 API에서 최신 일정 동기화 버튼
- 학교 추가/삭제

## 범위 밖 (NOT in scope)

- 보강 수업 등록/관리 (디자인에는 있지만 이 모듈에서는 **학사일정 조회만**)
- 학생 관리, 수업 관리 (다른 모듈 담당)
- 사용자 인증/로그인 (혼자 사용)
- 외부 배포 (로컬 전용)

## 사용자

- 학원 원장 1인 (로컬 PC에서 사용)

## 최종 통합 대상

- `wawa_smart_erp` 모노레포의 `apps/desktop/src/modules/school-calendar/`로 통합 예정
- 통합 전까지는 독립 실행 가능한 모듈로 개발

## 참고 디자인

프로젝트 루트에 2개의 HTML 디자인 시안이 제공되어 있음:
- **페이지 1**: 통합 캘린더 뷰 (학교 토글 + 월간 캘린더)
- **페이지 2**: 학교별 학사일정 관리 (학교 목록 + 일정 테이블)

디자인 시안의 HTML/Tailwind 코드를 최대한 재활용하여 React 컴포넌트로 변환.
